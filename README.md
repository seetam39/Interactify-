Interatify web app where we can interact with other users.
It has feaature of chatbot which is made by using Gemini API.
we can also video call with one or more users by using zegocloud .

Steps to installation .

1. Install Vs code or any other code editor.
2. Install Nodejs in yoor System .
3. Open this folder in your vs code .
4. Intall npm i
5. Install Pachages npm install express dotenv cookie-parser bcryptjs moongoose socket.io jsonwebtcker. (You can copy this and paste in your terminal )
6. To run frontend go to terminal and type cd\frontend. and then type npm run dev and run your website on http://localhost:8000 .
7. To run backend go to terminal and type npm run server .
8. You can also type npm start only and run your website on http://localhost:5000 .
